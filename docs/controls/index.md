# Controls

Every control takes `key`, `label`, and an optional `info` string shown as a tooltip on hover.

| control | description |
|---|---|
| `slider` | full-width range with scrubable readout |
| `toggle` | on/off switch |
| `color` | hex input, expandable with modes |
| `select` | dropdown |
| `stepper` | +/- buttons |
| `inputtext` | free-form text |
| `point2d` | draggable XY pad |
| `button` | action trigger |
| `text` | section separator |
| `folder` | collapsible group |
| `tabs` | tabbed sections |
| `link` | external URL button |
| `popup` | modal dialog trigger |
