# toggle

On/off switch.

```js
{ type: 'toggle', key: 'wireframe', label: 'wireframe', default: false }
```

| property | type | description |
|---|---|---|
| `key` | string | state key |
| `label` | string | display label |
| `default` | boolean | initial value |
| `info` | string | tooltip on hover |
