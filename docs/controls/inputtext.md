# inputtext

Free-form text input.

```js
{
  type:        'inputtext',
  key:         'title',
  label:       'title',
  default:     'untitled',
  placeholder: 'enter text...'
}
```

| property | type | description |
|---|---|---|
| `key` | string | state key |
| `label` | string | display label |
| `default` | string | initial value |
| `placeholder` | string | placeholder text |
